
lazbuild [options] <project/package filename or package name>

Parâmetros:

-?, -h, --help
                This help message.

-B, --build-all
                Build all files of project/package/IDE.

-r, --recursive
                Apply build flags (-B) to dependencies too.

-d, --skip-dependencies
                Do not compile dependencies.

--build-ide=<options>
                Build IDE with packages. Optional compiler options will be 
                passed after the options from used build mode and can be 
                specified here or with the --opt option.

-v, --version
                Show version and exit.

-q, --quiet
                Be less verbose. Can be given multiple times.
                Passing --quiet two times will pass -vw-n-h-i-l-d-u-t-p-c-x- to 
                the compiler.

--verbose
                Be more verbose. Can be given multiple times.

--verbose-pkgsearch
                Write what package files are searched and found.

--add-package
                Add package(s) to the list of installed packages (combine with 
                --build-ide to rebuild IDE).

--add-package-link=<.lpk file>
                Apenas registrar os arquivos de pacote Lazarus (.lpk). Não 
                construir.

--create-makefile
                Instead of compiling a package create a simple Makefile.

--pcp=<path>, --primary-config-path=<path>
                Primary config directory where Lazarus stores its config files. 
                Default is "C:\Users\fabio\AppData\Local\lazarus".

--scp=<path>, --secondary-config-path=<path>
                Secondary config directory where Lazarus searches for config 
                template files. Default is "C:\lazarus".

--os=<operating-system>, --operating-system=<operating-system>
                Override the project operating system. For example: win32 
                linux. Default: win64.

--ws=<widgetset>, --widgetset=<widgetset>
                Override the project widgetset. For example: gtk gtk2 qt win32 
                carbon. Default: win32.

--cpu=<cpu>
                Override the project CPU. For example: i386 x86_64 powerpc 
                powerpc_64. Default: x86_64.

--subtarget=<subtarget>
                Override the project subtarget.

--bm=<project/IDE build mode>, --build-mode=<project/IDE build mode>
                Override the project or IDE build mode.

--opt=<extra-options>
                Pass additional options to the compiler. Can be given multiple 
                times. If compilation options are also specified in 
                --build-ide, then the options from --opt will be added after 
                them.

--compiler=<ppcXXX>
                Override the default compiler. For example: ppc386 ppcx64 
                ppcppc. Default value is stored in environmentoptions.xml.

--language=<Language ID>
                Override language. For possible values see files in the 
                "languages" directory. Example: "--language=de".

--lazarusdir=<Lazarus directory>
                Directory to be used as a basedirectory.

--max-process-count=<count>
                Número máximo de "threads" para compilação em paralelo. Padrão 
                é 0, que tenta identificar o número de núcleos do sistema.

--no-write-project
                Não gravar arquivo informação do projeto atualizado após 
                construção. Se não especificado, número de construção será 
                incrementado se configurado.

--get=<text>, --get-expand-text=<text>
                Print the result of substituting macros in the text. The 
                absence of macros means the name of the macro. In case of an 
                error, returns only the text with partially expanded macros and 
                sets the error code (also for an empty string). Takes into 
                account the active (or specified via --bm option) build mode.

--get-build-modes
                Print a list of build modes in the project. Active mode is 
                listed first.
